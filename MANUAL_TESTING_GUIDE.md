# دليل الاختبار اليدوي - نظام أكنان العقارية

## المتطلبات الأساسية

- متصفح حديث (Chrome, Firefox, Safari, Edge)
- اتصال بالإنترنت
- حساب إدارة (للوصول إلى صفحات الإدارة)

## قائمة الاختبار اليدوي

### 1. الصفحة الرئيسية (`/`)

#### 1.1 التحميل الأولي
- [ ] الصفحة تُحمّل بدون أخطاء
- [ ] Header يظهر بشكل صحيح
- [ ] Footer يظهر بشكل صحيح
- [ ] لا توجد أخطاء في Console

#### 1.2 العقارات المميزة (Carousel)
- [ ] Carousel يعرض العقارات بشكل صحيح
- [ ] زر "السابق" يعمل
- [ ] زر "التالي" يعمل
- [ ] الصور تُحمّل بشكل lazy
- [ ] Blur placeholder يظهر قبل تحميل الصورة

#### 1.3 الأقسام
- [ ] قسم Vision يظهر بشكل صحيح
- [ ] قسم Mission يظهر بشكل صحيح
- [ ] قسم Values يظهر بشكل صحيح
- [ ] الروابط في Header تعمل

### 2. صفحة العقارات (`/properties`)

#### 2.1 عرض القائمة
- [ ] قائمة العقارات تُعرض بشكل صحيح
- [ ] PropertyCard يعرض البيانات بشكل صحيح
- [ ] الصور تُحمّل بشكل lazy
- [ ] Pagination يعمل (إن وجد)

#### 2.2 الفلاتر
- [ ] فتح/إغلاق الفلاتر يعمل
- [ ] اختيار المدينة يعمل
- [ ] اختيار الحي يعمل
- [ ] اختيار النوع يعمل
- [ ] اختيار المعاملة يعمل
- [ ] اختيار الحالة يعمل
- [ ] زر "تطبيق الفلاتر" يعمل
- [ ] زر "إلغاء" يعيد الفلاتر
- [ ] زر "مسح الفلاتر" يعمل

#### 2.3 البحث
- [ ] البحث في العنوان يعمل
- [ ] النتائج تُحدث بشكل صحيح

### 3. صفحة تفاصيل العقار (`/properties/[slug]`)

#### 3.1 البيانات الأساسية
- [ ] العنوان يظهر بشكل صحيح
- [ ] السعر يظهر بشكل صحيح
- [ ] الموقع يظهر بشكل صحيح
- [ ] الحالة تظهر بشكل صحيح

#### 3.2 الصور
- [ ] الصورة الرئيسية تُحمّل مع priority
- [ ] صور الـ gallery تُحمّل بشكل lazy
- [ ] Blur placeholder يظهر قبل تحميل الصورة
- [ ] Fallback placeholder يظهر عند فشل تحميل الصورة

#### 3.3 الخريطة
- [ ] الخريطة تُحمّل بشكل صحيح
- [ ] Marker يظهر في الموقع الصحيح
- [ ] Zoom controls تعمل

#### 3.4 التواصل
- [ ] زر واتساب يعمل
- [ ] زر الاتصال يعمل
- [ ] الرسالة تحتوي على بيانات العقار

### 4. صفحة المشاريع (`/projects`)

#### 4.1 القائمة
- [ ] قائمة المشاريع تُعرض بشكل صحيح
- [ ] Pagination يعمل
- [ ] النقر على المشروع يفتح صفحة التفاصيل

### 5. صفحة تفاصيل المشروع (`/projects/[id]`)

#### 5.1 البيانات
- [ ] معلومات المشروع تُعرض بشكل صحيح
- [ ] الإحصائيات تُعرض بشكل صحيح

#### 5.2 الخريطة
- [ ] الخريطة تُحمّل بشكل صحيح
- [ ] حدود المشروع تُعرض بشكل صحيح (أسود)
- [ ] القطع تُعرض بألوان صحيحة:
  - أخضر: متاح
  - أصفر: محجوز
  - أحمر: مباع
- [ ] النقر على القطعة يعرض InfoWindow
- [ ] النقر على حدود المشروع يعرض معلومات المشروع

### 6. الخريطة التفاعلية (`/map`)

#### 6.1 التحميل
- [ ] الخريطة تُحمّل بشكل صحيح
- [ ] لا توجد أخطاء في Console

#### 6.2 العرض
- [ ] العقارات تظهر كـ markers
- [ ] المشاريع تظهر كـ boundaries
- [ ] القطع تظهر كـ polygons بألوان صحيحة

#### 6.3 التفاعل
- [ ] النقر على marker يعرض InfoWindow
- [ ] النقر على polygon يعرض المعلومات
- [ ] الفلاتر تعمل بشكل صحيح
- [ ] البحث في الخريطة يعمل

### 7. صفحات الإدارة

#### 7.1 تسجيل الدخول (`/login`)
- [ ] صفحة تسجيل الدخول تُحمّل
- [ ] تسجيل الدخول بنجاح
- [ ] إعادة التوجيه إلى `/admin`

#### 7.2 لوحة التحكم (`/admin`)
- [ ] الإحصائيات تُعرض بشكل صحيح
- [ ] الروابط تعمل

#### 7.3 إضافة عقار (`/admin/properties/new`)
- [ ] النموذج يظهر بشكل صحيح
- [ ] LocationPicker يعمل
- [ ] رفع الصور يعمل
- [ ] حفظ العقار يعمل

#### 7.4 تعديل عقار (`/admin/properties/[id]/edit`)
- [ ] البيانات تُحمّل بشكل صحيح
- [ ] LocationPicker يعرض الموقع الحالي
- [ ] تحديث البيانات يعمل
- [ ] حذف العقار يعمل

### 8. الأداء

#### 8.1 Core Web Vitals (استخدم Lighthouse)
- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1
- [ ] FCP < 1.8s

#### 8.2 Network Performance
- [ ] عدد الطلبات معقول (< 50)
- [ ] حجم البيانات معقول (< 2MB)
- [ ] الصور تُحمّل بشكل تدريجي

### 9. التوافق

#### 9.1 المتصفحات
- [ ] Chrome (آخر إصدارين)
- [ ] Firefox (آخر إصدارين)
- [ ] Safari (آخر إصدارين)
- [ ] Edge (آخر إصدارين)

#### 9.2 الأجهزة
- [ ] Desktop (1920x1080)
- [ ] Tablet (768x1024)
- [ ] Mobile (375x667)
- [ ] Mobile Large (414x896)

### 10. الأخطاء

#### 10.1 Network Errors
- [ ] رسالة خطأ واضحة عند فشل Firebase
- [ ] رسالة خطأ واضحة عند فشل Google Maps
- [ ] Fallback مناسب عند فشل تحميل الصور

#### 10.2 Edge Cases
- [ ] عقار بدون صور يُعرض بشكل صحيح
- [ ] مشروع بدون قطع يُعرض بشكل صحيح
- [ ] بحث بدون نتائج يُعرض رسالة مناسبة

## كيفية الاستخدام

1. افتح التطبيق في المتصفح
2. انتقل عبر كل صفحة في القائمة
3. تحقق من كل بند في القائمة
4. سجّل أي مشاكل أو ملاحظات
5. استخدم Lighthouse لقياس الأداء

## أدوات مفيدة

- **Chrome DevTools**: Network, Performance, Console
- **Lighthouse**: Performance, Accessibility, SEO
- **React DevTools**: Component tree, Props, State
- **WebPageTest**: Performance testing from different locations

## تسجيل النتائج

استخدم هذا القالب لتسجيل النتائج:

```
الصفحة: [اسم الصفحة]
التاريخ: [التاريخ]
المتصفح: [المتصفح والإصدار]
النتيجة: ✓ / ✗ / ⚠️
الملاحظات: [أي ملاحظات]
```

**ملاحظة**: هذا دليل اختبار يدوي. للاختبارات الآلية، استخدم `npm run test`.
